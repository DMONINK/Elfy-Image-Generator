What Elfy can do
🗨️

Conversational memory

Each channel keeps its own context across sessions using local storage — Elfy remembers the conversation history.

🎨

Image generation

Text-to-image via Pollinations.ai Flux model with automatic prompt enhancement by Gemini.

✏️

Image editing

Attach a photo and ask Elfy to restyle it — anime, cartoon, oil painting — powered by Gemini's native image model.

📎

Multimodal inputs

Accepts images, audio, PDFs, code files, and CSVs directly in Discord attachments.

🎭

Custom personas

Use /forget [persona] to reset context and give Elfy a new personality on the fly.

🧵

Thread support

/createthread spawns a dedicated AI thread where Elfy responds to every message automatically.

How it works
1

Receive message

Elfy listens for DMs, @mentions, or any message in tracked channels/threads.

2

Detect intent

Keyword matching decides: image edit, image generate, or standard text chat.

3

Generate response

Gemini handles text and image editing. Image prompts go to Pollinations.ai Flux for generation.

4

Reply in Discord

Text is split to fit Discord's 2000-char limit. Images are sent as file attachments inline.

Self-host in minutes

# Clone & install git clone https://github.com/YOUR_USERNAME/Discord-Gemini-Chatbot cd Discord-Gemini-Chatbot pip install -r requirements.txt # Create .env and add your keys (see table below) python main.py
Variable Description
DISCORD_BOT_TOKEN required Your bot token from discord.com/developers
GOOGLE_AI_KEY required Gemini API key from ai.google.dev (free tier)
POLLINATIONS_API_KEY Your Pollinations.ai key for authenticated image generation

Powered by pollinations.ai

Image generation is routed through the Pollinations.ai gen.pollinations.ai API using the Flux model. Gemini first expands the user's short prompt into a richly detailed description, then Pollinations renders the final image — combining two AI systems for better results.
