What it does

One bot, many powers

Elfy brings together a friendly AI personality, per-channel memory, and creative image generation — all triggered naturally in chat.

💬

Natural Conversation

Powered by Gemini, Elfy keeps context across your whole channel. Talk to her by mentioning her or in DMs.

🎨

AI Image Generation

Ask Elfy to "draw", "generate", or "create" any image and get a stunning result via the Pollinations.ai Flux model.

🖼️

Image Editing

Upload a photo and ask Elfy to restyle it — turn it into anime, a painting, a cartoon, or any creative style.

📎

File & Media Support

Send images, audio, and documents. Elfy understands multimodal input thanks to Gemini 1.5 Flash.

🧠

Per-Channel Memory

Each Discord channel and thread has its own conversation history, persisted between bot restarts.

🎭

Custom Persona

Use /forget [persona] to give Elfy a new personality on the fly, or reset her back to her warm default self.

Slash commands

Everything Elfy can do

Beyond natural chat, Elfy has a handful of commands for managing context and threads.

Command Description
@Elfy [anything] Mention Elfy in any channel to chat, ask questions, or request images.
DM Elfy Slide into her DMs for a private conversation with full memory.
/forget Clear channel history and reset to default personality.
/forget [persona] Clear history and give Elfy a new custom persona instantly.
/createthread [name] Create a dedicated thread where Elfy responds to every message.
generate image of… Natural language trigger — just describe an image in chat.
[upload image] + "turn this into…" Attach a photo and ask Elfy to restyle or transform it.
Get started

Run Elfy in your server

Self-hosted and open source. You'll need a Google Gemini key and a Discord bot token.

1

Clone the repository

Download the source or git clone the repo to your machine.

2

Get your API keys

Grab a free Gemini key from ai.google.dev and create a Discord bot at discord.com/developers. Optionally grab a Pollinations.ai key at pollinations.ai.

3

Create your .env file

Add GOOGLE_AI_KEY, DISCORD_BOT_TOKEN, and optionally POLLINATIONS_API_KEY.

4

Install & run

pip install -r requirements.txt then python main.py — Elfy is online!
